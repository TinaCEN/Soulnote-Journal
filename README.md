## Development Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Install Expo CLI:
   ```bash
   npm install -g expo-cli
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Use Expo Go app on your iOS device to scan the QR code and test the app
